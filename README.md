# Todo App with Context API
A simple React Todo App built using Context API

## Features
- Add todos
- Delete todos
- Edit todos
- Mark todos as completed
- Filter todos
- Clear completed todos
- Light and dark theme toggle
- Save todos and theme in localStorage

## Concepts Used

- React
- TypeScript
- Context API
- useState
- useEffect
- localStorage

# How to run
 npm install

 # Summary 
 This app will use Context API to manage todo state, filter state, and theme state. Todos and theme are saved in localstorage, so data stays after after page refresh.
